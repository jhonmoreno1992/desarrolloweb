HTML

HTML son las siglas de HyperText Markup Language. Es el lenguaje de marcado principal para la creacion de paginas web.

Creado por Tim Berners-Lee, HTML ha evolucionado significativamente desde sus inicios.

Historia de HTML

Los inicios del lenguage HTML se remotan a principios de la decada de 1990. Desde entonces, ha pasado por varias versiones, cada una añadiendo nuevas funcionalidades y mejoras.

Versiones de HTML

Tim Berners-Lee definio la primera version de HTML en el año 1991.

En la actualidad, la ultima version de HTML es HTML5.
